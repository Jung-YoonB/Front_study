웹 스토리지

브라우저 내부에 데이터를 키-값(key-value) 쌍으로 저장하는 공간
    - localStorage : 영구 저장 (브라우저나 탭을 닫아도 데이터가 유지 됨)
    - sessionStorage : 택 단위 저장 (해당 탭을 닫으면 데이터가 삭제 됨)

주요 메소드 : setItem(ket, value), getItem(key), removeItem(key), clear()
[참고] 웹 스토리지는 "문자열" 형태의 데이터만 저장할 수 있음

localStorage

브라우저를 닫거나 다시 열어도 유지 되어야 하는 데이터에 사용
    - 사용자 설정 : 다크모드/라이트모드, 언어 설정 등
    - 비회원 장바구니 : 로그인 전 장바구니 데이터를 임시 보관
    - 오늘 하루 이 창 보지 않기 : 팝업창 만요 시간 및 상태 저장

sessionStorage

데이터가 영구적으로 남으면 안 되거나, 현재 탭이 유지되는 동안만 일시적으로 보관할 때 사용
    - 작성 폼 백업 : 게시글 작성 중 실수로 새로고침을 해도 작성 중인 내용 유지
    - 목록 상태 유지 : 상품 목록 스크롤 위치 및 필터 조건을 유지하면서 상세 페이지에서 돌아왔을 때 복원
