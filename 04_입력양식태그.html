폼(form) 태그>

폼 태그 내에 submit 버튼 클릭 시, 해당 폼 태그 내에 작성 된
사용자가 입력 한 값들을 서버로 전달하면서 요청을 수행

속성

  • action : 요청 할 서버 주소(경로)
  • method : 요청 전송 방식
    • get 방식 : 요청 시 사용자가 입력한 값들이 URL에 노출되는 방식 (기본값)
    • post 방식 : 요청 시 사용자가 입력한 값들이 URL에 노출되지 않는 방식
